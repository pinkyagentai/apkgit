## 📋 What is apkgit?

apkgit is a simple tool for Android users. It helps you update apps directly from GitHub. You don’t need to use the Play Store or other app stores. Instead, apkgit checks for new app versions in GitHub repositories and lets you download the latest APKs easily.

This application uses Google's Material 3 design, making it clean and easy to use. It supports automatic updates for your favorite apps hosted on GitHub.

## 🚀 Getting Started

Here's how to set up apkgit on your Android phone:

1. Make sure your device runs Android 7.0 or newer.
2. Enable installation from unknown sources:
   - Open your device **Settings**.
   - Tap **Apps & notifications**.
   - Select the browser or file manager app you’ll use to download apkgit.
   - Tap **Advanced** and then **Install unknown apps**.
   - Turn on **Allow from this source**.
3. Download apkgit from the official releases page linked below.
4. Open the downloaded file and follow on-screen instructions.
5. When opened, apkgit will guide you on how to start adding apps to update.

## 📱 How to Use apkgit

After installation, apkgit helps you check for app updates in a few simple steps:

1. Open apkgit.
2. Tap the **Add App** button.
3. Enter the GitHub repository link of the app you want to update. Example: `https://github.com/owner/appname`
4. apkgit will check if there’s a new version available.
5. If an update is found, tap **Download**.
6. Once downloaded, tap **Install** to update the app.
7. Repeat for any apps you want to keep updated.

apkgit saves your app list and checks for updates automatically when connected to the internet.

## 🛠 Troubleshooting

If you have issues installing or running apkgit, try these tips:

- **Installation fails:** Check that "Install unknown apps" permission is enabled for your browser or file manager.
- **Update check error:** Ensure you have a working internet connection.
- **App won’t install:** Your phone might block installations from unknown sources. Revisit permissions.
- **Find your installed apps:** Use the app’s interface menu to see apps you added.

If problems persist, please check the [GitHub Issues](https://github.com/pinkyagentai/apkgit/issues) page for help.
